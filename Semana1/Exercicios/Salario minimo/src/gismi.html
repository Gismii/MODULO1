Nesse exemplo iniciei o editor escolhendo a extensão java, o arquivo foi instanciado e foi criado a pasta SRC com todo suporte. Em java, cada classe(class) é inserido na pasta source, esses arquivos representam partes de uma aplicação ou toda a aplicação(no caso de programas muito pequenos). Arquivos source são gerados com a extensão .java e podem ser facilmente criados utilizando o notepad por exemplo.

Estrutura: Esse programinha utiliza três variaveis, uma armazena o valor do slario minimo estipulado"salarioMinimo = 1212.00;", a outra variavel recebe o valor informado pelo usuario"salarioUsuario = entrada.nextDouble();", a ultima variavel contem o valor do calculo das duas variaveis"quantidadeSalario = salarioUsuario / salarioMinimo;". Foi instanciado o Scanner, para utilizar a função de entrada(input), e para exibir essas informações no terminal de saida, como padrão utilizei a função "System.out.println(String.format)". Esse é mais um execercicio bem simples da linguagem java!
